# Bank-Management-System
The tasks that admin can perform: 
   •     Add user 
   •     Delete user 
   •     Edit username/password 
   •     View user log 
The tasks that staff can perform: 
   •     Create new account 
   •     Cash Deposit 
   •     Cash withdrawal 
   •     Fund Transfer 
   •     Account information 
   •     Transaction information 
   •     Log out
